# Part B – Essential Linux Commands

Several basic Linux commands were executed to understand navigation and system information.

# Commands Used

# `pwd`
Displays the present working directory.

# `ls`
Lists all files and directories in the current location.

# `ls -la`
Displays detailed information about files and folders, including hidden files.

# `cd`
Changes the current directory.

# `clear`
Clears the terminal screen.

# `history`
Displays previously executed terminal commands.

# `whoami`
Shows the username of the currently logged-in user.

# `hostname`
Displays the system hostname.
