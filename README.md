## 💻 Part B: Basic Navigation Commands
Below is the documentation for the essential navigation commands executed during this task[cite: 1].

| Command | Purpose |
| :--- | :--- |
| `pwd` | **Print Working Directory:** Displays the absolute path of the current directory you are in[cite: 1]. |
| `ls` | **List:** Lists the files and directories inside the current working directory[cite: 1]. |
| `ls -la` | **List Long & All:** Lists all files (including hidden ones starting with a `.`) with detailed attributes like permissions, size, and owner[cite: 1]. |
| `cd` | **Change Directory:** Used to navigate between different folders/directories in the file system[cite: 1]. |
| `clear` | **Clear Screen:** Clears all previous commands and outputs from the active terminal screen viewport[cite: 1]. |
| `history` | **Command History:** Lists the sequence of commands previously executed in the terminal session[cite: 1]. |
| `whoami` | **Who Am I:** Prints the username of the current active user logged into the shell[cite: 1]. |
| `hostname` | **Host Name:** Displays the network name assigned to the local host/machine system[cite: 1]. |
